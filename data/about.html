INTRODUCTION

The advancement of the Internet of Things (IoT) and embedded systems has opened new possibilities in human–machine interaction, robotics, and remote control technologies. This project, titled "IoT Robotic Hand", aims to design and implement a robotic hand that can be wirelessly controlled through both a sensor-equipped glove and a web-based interface.

The project utilizes two ESP32 microcontrollers—one embedded in the robotic hand and another integrated into a wearable glove. The glove is equipped with flex sensors that detect the bending motion of each finger. These sensor readings are transmitted via Wi-Fi to the ESP32 controlling the robotic hand, which replicates the same movements in real time using servo motors. This setup allows the robotic hand to imitate natural human gestures with high accuracy and low latency.

In addition to gesture-based control, the system also features a web server interface hosted on the ESP32, allowing users to control the robotic hand remotely through a standard web browser.

The IoT Robotic Hand project serves as a prototype for future applications in assistive technology, teleoperation, prosthetics, and remote robotic systems. It showcases how affordable hardware and IoT connectivity can be combined to create interactive and practical solutions in modern robotics.

PROGRAMMING

Backend — C++

Implements an onboard web server using C++ (Arduino Framework) to handle HTTP requests, control servo motors, and manage communication between the browser, robotic hand & sensor glove.

  • Handles Wi-Fi and AP modes
  • Controls hardware (GPIO, PWM, Serial)
  • Can log commands

Frontend — HTML, CSS, JavaScript

Provides a responsive web interface for controlling the robotic hand from any browser. JavaScript sends commands to the ESP32’s server to move servos and display feedback dynamically.

  • Real-time control with sliders & buttons
  • Communicates via HTTP requests
  • Displays live feedback and status

Database — MongoDB

Stores command logs sent to the ESP32, including timestamps and results, enabling traceability and data analytics.

  • Schemaless JSON-like storage
  • Ideal for simple IoT logging
